Art Pieces Created
Using Computer Aided Design

I have always had a passion for how design applies to art and the tools required to do so. This started when I was young and has since forth culminated into a life-long interest in the arts. Be it woodworking, sculpture, pottery, music, painting, drawing, photography, or designing things on the computer, I have always had a passion for creating art. The most recent medium that has sparked my interest was the use of various computer softwares to create art. The majority of the pieces shown below were created using the computer aided design (CAD) software Solidworks in order to create models of my various ideas and inspirations using a variety of applications such as surface tools, sheetmetal, motion, etc. Renders were created using Solidworks Visualize. The piece above was created using Solidworks in order to create each individual part to then be put into an assembly. The piece is designed to be an outdoor wind sculpture and was inspired by the works of Anthony Howe and Lyman Whitaker. The animation was created using Motion within Solidworks.
